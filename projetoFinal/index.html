Explicação de segurança
Hash + Salt
bcrypt com cost 12 ≈ 300ms/hash. Salt único via crypto.randomBytes impede rainbow tables. Hash não é reversível.

TOTP (2FA)
Código HMAC-SHA1 de 6 dígitos válido por 30s. Tolerância de ±1 passo para drift de relógio. Replay prevention: cada código usado uma vez.

Sessão
Token de 256 bits armazenado server-side. Cookie httpOnly + SameSite. TTL absoluto de 2h + idle timeout de 30min.

Bloqueio
5 falhas consecutivas bloqueiam por 15min. Resposta genérica evita enumeração de e-mails válidos.
